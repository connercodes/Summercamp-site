  • Coding Camp

    Learn basics of programming. Students will discover Object Oriented programming in Java to create games and applications. Learn processing commands while also using creativity to create original applications. Ages 12-17.

  • Robotics Camp

    Engineering campers will build robot which can perform tasks. As a team, campers will develop skills in programming, problem solving and engineering. Three week course will be offered throughout the summer with a different theme for each session. Trophies will be given for best design.

  • STEM Camp

    Universities throughout Florida are now offering STEM camps for middle and high school students. The Florida Board of professional engineers has put together a list of colleges in Florida that are offering STEM Camps.

  • Video Game Design Camp

    Create your own video games using the same tools as industry pros. Design games, characters and levels using the Unity™ game engine – or learn how to create mods in DMA’s popular Minecraft™ camps and more.
